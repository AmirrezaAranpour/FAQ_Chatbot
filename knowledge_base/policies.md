# Policies

## Privacy
- Client data is used only to deliver the project
- Access is minimal and granted on a need-to-know basis

## Refunds (Fixed Price)
- If work has not started: full refund of the first milestone is possible
- If work has started: refund is calculated based on completed work

## Rescheduling
- Meetings can be rescheduled with at least 24 hours notice


---
## Cancellation (Fixed Price)
If you need to cancel a Fixed Price project, the refund follows the same rule as the Fixed Price refunds above:
- If work has not started: full refund of the first milestone is possible
- If work has started: refund is calculated based on completed work
